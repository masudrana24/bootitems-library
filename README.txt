== Description ==

A plugin made by bootitems.com to extends its free themes features by adding functionality to import demo data content in just a click.
This plugin requires the [One Click Demo Import](https://wordpress.org/plugins/one-click-demo-import/) plugin to be installed.


How to use this Bootitems Library?

1. While installing free themes by bootitems do kindly install & activate our plugin "Bootitems Library " or you can do this manually too.
2. To import demo contents, please go to Appearance > Import Demo Data. 
3. Choose your demo by checking live preview and then click import button to import the demo.
4. You can delete this plugin once you're done with importing.
5. Feel free to contact bootitems support team at ( https://bootitems.com/contact/ ) if you need any assistance.

== Installation ==

*** From your WordPress dashboard ***

1. Visit 'Plugins > Add New',
2. Search for 'Bootitems Library' and install the plugin.
3. Activate 'Bootitems Library' from your Plugins page.


== Frequently Asked Questions ==

= Does this plugin works with any WordPress theme =

No, This plugin will only work with the themes developed by [Bootitems](https://bootitems.com/).
